# VCT-13: Meeting detail — add note form

**Type:** story
**Status:** todo

As a user, I want to add a note to a meeting from the detail page so that I can capture discussion as it happens. Depends on VCT-3, VCT-12.

**Acceptance criteria:**
- [ ] A form on the meeting detail page lets the user add a note (author + text) via `MeetingsService`
- [ ] On success, the notes feed updates without a full page reload
- [ ] Loading/error state shown for the submission itself
